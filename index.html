Neonano: Ayanami Bot

What is Ayanami?

Ayanami is a Discord bot created for the main purpose of helping with moderation. It also introduces a server economy as an activity with more to come in the future. This is a completely self hosted bot. Data should stay constant even if Ayanami has to restart or goes down for a bit.


What are the commands?

  • ping - Pings Ayanami to check if it's online and for the latency.
  • mute - Mutes the user for a set period of time. s for seconds, m for minutes, h for hours, then d for days.
  • unmute - Unmutes the muted user.
  • kick - Kicks out a user from the server.
  • ban - Bans a user from the server.
  • unban - Unbans a user from the server.
  • warn - Adds a warning to a list about that user.
  • rwarn - Removes a warning about that user from the list.
  • setlog - Sets up the channel the command was sent in as the channel for logging deleted/edited messages. Only people with admin permissions can use this command.
  • purge - Mass removes a set of messages that you choose.

Are there any fun commands?

  • work - Can work for coins.
  • steal - Can steal coins from others.
  • bal - Checks your balance.
  • mitalol - Gambles your coins away. You can pick a number 1-9 and pick a color between red, black, and green.
  • evaquote - Sends a random Neon Genesis Evangelion quote.

Common issues

  • Bot doesn't have permission to mute this user - This bot must be higher a role than the role that person has.
  • Application did not respond - I would suggest doing the command with the "!" prefix if that happens. Example: !mute @user 30m spamming
  • Ayanami isn't logging the messages - Please make sure she can view and send messages in that channel. And if you didn't already, do /setlog or !setlog to set the channel she logs messages in.
